# 🛡️ War Room BAaaS
**Custom Operations & Profitability Command Center for EPC Firms**

War Room BAaaS (Business Automation as a Service) is a modular, Python-powered dashboard designed to bridge the gap between field operations and the executive boardroom in mid-sized Engineering, Procurement, and Construction (EPC) companies.

By ingesting standard daily site logs, the War Room provides real-time, military-grade financial insights, shifting project oversight from reactive troubleshooting to proactive profit protection.

## 📊 Core Modules

1. **Material Leakage Command:** * Tracks Actual vs. Estimated consumption of high-value items (Steel, Cement, Fuel).
   * Automatically calculates exact financial loss from site-level material variances.
2. **Machinery Utilization Tracker:**
   * Monitors Working vs. Idle hours across heavy equipment fleets (Cranes, JCBs, DGs).
   * Converts idle hours into "Sunk Cost" financial metrics based on rental and operator rates.
3. **Manpower Efficiency Matrix:**
   * Analyzes the Indirect (Overhead) vs. Direct (Production) labor ratio.
   * Tracks daily payroll burn rates and site staffing distributions.

## 🛠️ Technology Stack
* **Logic Engine:** Python 3.10+
* **Data Processing:** Pandas, NumPy
* **Frontend UI:** Streamlit
* **Interactive Visualization:** Plotly Graph Objects & Express
